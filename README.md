# Address-Book
Hands-on experience with real-world C programming.
Understanding data management using structures.

# Description
Project demonstrates the use of structures, pointers, file handling, and modular programming to manage contact information efficiently.
The application allows users to store, view, search, edit, and delete contact details such as name, phone number, and email address. Data persistence is achieved using file operations, ensuring that contact information is retained even after program termination.
This project is designed to strengthen core C programming concepts commonly used in embedded software development.
